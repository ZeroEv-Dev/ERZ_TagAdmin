# ERZ_TagAdmin

TAG ADMIN IN GAME

## INSTALLAZIONE
- INSERISCI IN `[esx]`
- AGGIUNGI QUESTO NEL TUO `server.cfg`:

```
start erz_tagadmin
```

### USO
Comando per Attivare/Disattivare il Tag: /tag
